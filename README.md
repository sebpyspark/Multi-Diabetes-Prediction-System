# Multi-Diabetes-Prediction-System - 13 Types
AI-powered diabetes prediction platform capable of predicting 13 different types of diabetes using TensorFlow, Scikit-learn, ipywidgets, HTML, and CSS.

How to Execute this project-Follow the steps below

0. Click Open in Colab link-->in COLAB click Run All Option
1. The user can Click option 6.Single Input Predictions directly by putting the clinical parameters listed in the customer GUI.
2. After entering your values click 'Save' button then click the 'Predict' button
3. For getting predictions using bulk input data the user can click option 7.Bulk Input Predictions. The input data for bulk prediction is already loaded in the CSV file, Bulk_Diabetes_Data.csv. This option i indented for hospitals to get a summary of different diabetic patients over a period of time
4. This model is created as per the E-T-L methodology. You can click option 1 thru 8 to create model or display output using the saved model here.
